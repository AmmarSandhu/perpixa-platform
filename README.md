# Perpixa Platform

Perpixa is a commercial AI platform built for pay-as-you-go automation tools.

## v1 Scope
- Video Generation AI
- Credit-based monetization
- Job-based execution
- Secure asset downloads

## Architecture
- Backend: FastAPI (Python)
- Frontend: Web Application
- Billing: Credit-based (no subscriptions in v1)

## Planned Extensions
- CAD AI
- Image Generation AI
